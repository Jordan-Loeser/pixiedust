## Config Values

| Key           | Type               | Description                                    | Default |
| ------------- | ------------------ | ---------------------------------------------- | ------- |
| `stationId`\* | `string \| number` | The station id according to [wtfmta]().        | n/a     |
| `direction`\* | `"N" \| "S"`       | The direction to show (aka Uptown or Downtown) | n/a     |
